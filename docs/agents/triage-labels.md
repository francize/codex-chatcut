# Triage labels

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | Reporter input is required |
| `ready-for-agent` | `ready-for-agent` | Fully specified and safe for an agent to claim |
| `ready-for-human` | `ready-for-human` | Human judgment or credentials are required |
| `wontfix` | `wontfix` | The repository will not action the request |
